About

Know More About Us

"Complaino" is a webpage of handling, managing, responding to, and reporting student and faculty grievances. This process requires a streamlined approach and constant monitoring to ensure faster resolutions. The system is made correctly, and all the testing is done as per the requirements. So, the rest of the things depend on the user, and no one can harm the data or the software if the proper care is done. All the attributes are working correctly, and if any error is found, then it can be removed easily. Complaino is based on web technology which aims to enable the submission of grievances by citizens from anywhere and to track their grievances through system generated unique registration number. This Application provides a centralised approach for recording and managing information and generating different kinds of reports.

Admin

Admin has the official powers to control the flow of the data from one part of the system to the other. He can manipulate the access of the users to the data.
